# Overview

组合检索工具, 用于查询符合条件的双色球彩票结果

+ 可筛选选取参数, 例如33选11
+ 可筛选奇数, 偶数, 质数, 合数, 连数出现的次数
+ 可筛选选择结果之和的范围
+ 可增加若干条件, 筛选特定数字出现的次数

# Build

```
git clone https://github.com/samlior/combinations-search-tool.git
cd combinations-search-tool
npm i
cd page-activate
npm i
npm run watch
cd ../page-agreement
npm i
npm run watch
cd ../page-main
npm i
npm run watch
cd ..
npm start
```
